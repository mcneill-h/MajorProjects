# Summary of my projects:

I have developed several projects covering **artificial intelligence**, **software engineering**, and **game development**. These include a **Physics-based** game inspired by Sisyphus, a Blackjack AI that uses **Deep Q-Learning** to learn optimal strategies, a **Minimax AI** for the Nine Men's Morris board game, and a Home Designer application using **Raycasting** to create virtual living spaces. Through these projects, I gained experience in algorithms, object-oriented programming, AI techniques, and user-focused software design.

In addition, I independently developed and published **Cat Chaos**, a commercial mobile game available on the App Store. From concept and implementation to release and maintenance, I managed the entire development process. 

Together, these projects showcase my ability to design, build, and deploy complete software products while working across game development, AI, and interactive applications.

A corresponding license is in each folder. Have fun!
